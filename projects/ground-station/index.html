Ground Control Station

Control all the things robotics.

robotics ros prototyping electronics python arduino jetson nano I/O CAD 3d printing laser cutting hackaday prize wireless wifibroadcast

Overview

After making the Cablecam, I saw the need to have a portable and customizable controller for that one and other robots to come. One unit that could integrate a computer, inputs and communications hardware would be beneficial for robotics development and deployment. Wanted to make something neat, and got inspiration from other professional GCS. I have always been fascinated by these.

This was an overkill for the use I’d give it, and was made purposely for future robots I would build, especially for an oncoming project that would benefit from it. Hopefully it can serve as inspiration and a base for people to build upon. I got to practice and learn a few things on the way.

This project was a finalist on the Hackaday Prize 2021, a global hardware hacking competition. It was one of the 10 finalists on the Redefine Robots category.

For more in depth information check this project out on Hackaday.io.
